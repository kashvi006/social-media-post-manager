# Social Post Composer

A React application built with Redux Toolkit that allows users to create, edit, delete, and filter social media posts.

## Features

- Add new posts
- Edit existing posts
- Delete posts
- Platform filtering
- Redux Toolkit state management
- React Hooks
- Responsive UI

## Technologies Used

- React.js
- Redux Toolkit
- React Redux
- CSS3

## Installation

```bash
npm install
npm start
